이 코드는 다음의 프로젝트의 일부임

프로젝트 소개: 
날씨에 따른 유동인구&소비 데이터를 분석하여 지역별 소비 패턴과 변동성을 파악하고, 창업 의사결정에 활용 가능한 인사이트를 제공하는 서비스 

핵심 아이디어: 
데이터 융합을 통한 상권 이해 고도화 
⇒ 인구 흐름(잠재 고객)과 실제 소비(실현된 매출), 날씨(외부 요인)를 함께 고려해 상권의 실질 경쟁력을 평가 

창업 의사결정 지원 
⇒ “이 지역에는 어떤 업종이 잘 될까?”, “날씨에 따라 매출이 얼마나 변동할까?” 등의 실질적인 질문에 답하는 데이터 인사이트 제공 

시각화 및 AI 질의응답 결합 
⇒ Power BI 기반 시각화와 GPT를 결합하여, 복잡한 데이터를 쉽게 해석하고 질의응답 형태로 인사이트 전달 

[ 프로젝트 목적 ] 
인구와 소비, 날씨의 상관관계 탐색 
⇒ 인구 유동성, 소비 데이터, 일자별, 기온·강수량 등과 상권 활발도의 관계 분석 

데이터 기반의 상권 인사이트 제공 
⇒ 지역별 인구 변동과 날씨 변동을 직관적으로 이해할 수 있도록 지원 

시각화 + AI 결합형 분석 서비스 구현 
⇒ Power BI를 통한 시각적 이해와 GPT형 질의응답을 결합하여 정보 접근성을 향상 

[ 주요 기능 ] 
데이터 분석 
- 인구 변화, 소비 데이터(날씨, 계절별) 분석 

시각화(Power BI) 
- 지역별 유동인구 추이 

날씨 변화에 따른 인구 증감 그래프 
날씨 변화에 따른 소비 변화 그래프 

시스템 모니터링: 
  • Azure Monitor & Application Insights 
  • 성능 추적, 에러 알림 

  
데이터 흐름 요약: 
  • 실시간: Functions → Event Hub → Stream Analytics → Azure SQL Database 
  • 배치1: 로컬 → ADLS Silver/Gold 
  • 배치2: Functions → ADLS Bronze → Data Factory → Azure SQL Database
